# Simulador de Adopción de Vehículos y Emisiones (FONDECYT)
Este proyecto replica y expande un modelo de dinámica de sistemas (originalmente en Vensim) utilizando Python. 
Evalúa la adopción de vehículos eléctricos (EV) vs combustión (ICEV), el impacto en emisiones de CO2 y los costos de políticas públicas en Chile.
